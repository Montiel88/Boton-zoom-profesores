# Zoom Monitor - Sistema de Monitoreo de Clases

Sistema para monitorear y auditar las reuniones de Zoom del Instituto Tecnológico San Antonio (TESA).

## Características
- Sincronización automática con API de Zoom
- Webhooks para datos en tiempo real
- Historial completo de reuniones
- Visualización de asistentes y grabaciones
- Gestión de usuarios y roles
- Exportación de reportes

## Instalación
Ver archivo [INSTALL.md](INSTALL.md) para instrucciones detalladas.

## Tecnologías
- PHP 8+
- MySQL
- JavaScript (vanilla)
- Zoom API (OAuth Server-to-Server)
